> **你的 AI 编程伙伴们各自忙碌，而 Hum 是那个帮你看着所有人的小家伙。**

Hum 的原型是灯塔水母（*Turritopsis dohrnii*）—— 地球上唯一永生的生物。它虽然小小的、透明的，却能在深海中感知一切，永远不会消失。

我们相信，当 AI 编程助手越来越多，你需要的不是又一个 AI，而是一个**懂你的小管家**。

---

## 🌊 现在 — Hum 已经能做到的事

| 能力 | 说明 |
|:---|:---|
| **6 个 AI Agent 接入** | Claude Code · Codex · QoderWork · Qwen Code · Gemini CLI · Kimi K1 |
| **语音播报** | 把 agent 事件变成自然语言播报，不用切窗口就知道发生了什么 |
| **权限审批** | 语音说"确认" / 键盘按 Y / 点击按钮 —— 三种方式批准 agent 操作 |
| **深海吸收态** | 每个 agent 化身一种深海生物，围绕 Hum 旋转 |
| **统计面板** | Token 用量、花费、工具调用 —— 一目了然 |
| **狂暴模式** | 全自动确认，agent 不再等你，全力冲刺 |

## 🐚 近期 — 即将到来（1-2 个月）

### 📱 手机远程控制

```
┌─────────────┐         ┌──────────────┐
│   你的电脑   │ ──LAN── │  你的手机     │
│             │         │  (浏览器)     │
│  🪼 Hum     │ ◄─────► │  ┌─────────┐ │
│  + Agents   │         │  │ Allow?  │ │
│             │         │  │ [✓] [✗] │ │
└─────────────┘         │  └─────────┘ │
                        └──────────────┘
```

- 扫码连接，无需安装任何 App
- 手机上审批权限、回答问题、查看 agent 状态
- 离开工位也不会错过任何请求

### 📊 Agent 对比面板

哪个 agent 最快？哪个最省钱？哪个用了最多工具？一个面板告诉你。

### 🔌 更多 Agent 接入

Cursor · Windsurf · Copilot · Aide · Augment …… 任何支持 hooks 的 AI 工具都能一行脚本接入。

### 🎨 Rive 动画升级

用 Rive 替换 Canvas2D，Hum 的动作更流畅、更有灵魂。

### 🔄 跨设备信息流转

> **所有应该流转于个人侧的东西，都应该能流转。**

- **偏好同步** —— 在公司电脑 allowAlways 了一个工具，家里的电脑自动生效
- **上下文漫游** —— 在手机上浏览电脑端 agent 的对话摘要，出门也能了解进度
- **通知聚合** —— 多台电脑、多个 agent 的通知汇聚到一个入口，不再遗漏

### 🧠 智能权限策略

- Hum 学习你的审批习惯，自动建议 allow 或 deny
- 按项目 / agent / 工具类型设置细粒度规则
- "这个项目里 Claude 的 Bash 命令我都信任" —— 一句话配好

### 🤝 多 Agent 协作编排

- Hum 作为调度者，把大任务拆分给不同 agent
- Agent 之间共享上下文，减少重复工作
- 你只需要和 Hum 说一句话，它帮你安排

## 🔭 远期 — 开放生态（6-12 个月）

| 方向 | 目标 |
|:---|:---|
| **插件系统** | 自定义事件处理器、TTS/STT 提供商、播报风格 |
| **Hook 协议标准** | 发布开放标准，让任何 AI 工具一行命令接入 HumHum 生态 |
| **团队模式** | 和同事共享 agent 状态、协作审批、团队仪表盘 |
| **人格系统** | 不同性格的 Hum —— 元气型、冷静型、毒舌型 —— 影响播报风格和互动方式 |
| **跨平台** | Windows · Linux 完整支持 |
