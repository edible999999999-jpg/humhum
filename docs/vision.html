你的偏好、权限、记忆 —— 全部在 Hum 这里汇合，跟着你走，不跟着设备。

  • 偏好同步 —— 在公司电脑 allowAlways 了一个工具，家里的电脑自动生效
  • 上下文漫游 —— 在手机上浏览电脑端 Agent 的对话摘要，出门也能了解进度
  • 通知聚合 —— 多台电脑、多个 Agent 的通知汇聚到一个入口，不再遗漏
  • 智能权限策略 —— Hum 学习你的审批习惯，自动建议 allow 或 deny
  • 多 Agent 协作编排 —— Hum 作为调度者，分发任务给不同 Agent
