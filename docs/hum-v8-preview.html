Three.js 体积渲染

MeshPhysicalMaterial + transmission 实现真正的半透明玻璃质感，光线穿过水母体时折射散射。

Canvas2D: 只能用 opacity 模拟，无法做折射

Bloom 辉光

UnrealBloomPass 后处理，高亮区域自然溢出发光。生物发光的核心视觉效果。

Canvas2D: globalCompositeOperation lighter 是近似，不是真 bloom

3D 粒子系统

BufferGeometry + PointsMaterial，数千粒子在三维空间中漂浮，有景深虚化。

Canvas2D: 粒子只有 2D 平面

顶点动画触手

CatmullRomCurve3 驱动 TubeGeometry，每帧更新控制点。触手在 3D 空间中真实飘荡。

Canvas2D: 只有贝塞尔曲线的 2D 投影
